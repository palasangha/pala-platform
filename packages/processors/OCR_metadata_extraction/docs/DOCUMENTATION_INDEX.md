### Setup Command (1 line)
```bash
export GOOGLE_APPLICATION_CREDENTIALS="/path/to/credentials.json" && \
export GOOGLE_LENS_ENABLED=true
```

### Test Command
```bash
python test_google_lens.py
```

### Simple Usage
```python
from app.services.ocr_service import OCRService
result = OCRService().process_image('letter.jpg', provider='google_lens')
print(result['metadata']['sender'])
```

### REST API
```bash
curl -X POST http://localhost:5000/ocr/process/IMAGE_ID \
  -H "Authorization: Bearer TOKEN" \
  -d '{"provider": "google_lens"}'
```
