## Essential Commands

### Development
```bash
# Setup
python3 -m venv rbac_test_env
source rbac_test_env/bin/activate
pip install -r backend/requirements.txt

# Run tests
pytest tests/test_rbac_unit.py -v

# Start backend
python backend/run.py

# Start frontend
cd frontend && npm start
```

### Deployment
```bash
# Apply fixes
python3 -m py_compile backend/app/routes/rbac.py
python3 -m py_compile backend/app/routes/dashboard.py

# Run tests
pytest tests/ -v --cov=backend/app

# Deploy
docker-compose up -d
```
