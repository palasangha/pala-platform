# Packages

This directory contains the core packages of the Pala Platform.

## Structure

- **processors/** - Document processing engines (OCR, PalaScribe)
- **mcp-server/** - MCP (Model Context Protocol) orchestrator
- **agents/** - AI agents for enrichment and processing
- **storage/** - Storage layer abstraction
- **enrichment/** - Metadata and digital signing services
- **shared/** - Shared utilities and types
